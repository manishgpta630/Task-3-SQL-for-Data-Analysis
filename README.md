Analyze the Ecommerce dataset using SQL queries to extract meaningful business insights.

- MySQL 8.0
- Ecommerce SQL Dataset

- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Subqueries
- Aggregate Functions (SUM, AVG)

3. Sort records using ORDER BY.

5. Perform INNER JOIN between tables.
6. Perform LEFT JOIN.
7. Perform RIGHT JOIN.
8. Execute Subqueries.
